## Installation dependencies
- Install MongoDB with enable transaction (version 5^)
- Install Redis (version 6^)
- Install Node (version 14^)

## Fill environment
- copy .env.example to .env and fill full information
## Installation packages
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Fix code style and formatting issues

```bash
npm run lint
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
